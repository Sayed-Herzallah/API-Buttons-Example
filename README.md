# 🖱️ API Buttons Example

A clean, interactive frontend web application showcasing how to dynamically fetch data from public APIs using interactive buttons and render it on the DOM in real-time.

---

## 📌 Overview

**API-Buttons-Example** is a practical tutorial-focused project demonstrating JavaScript asynchronous operations. It enables users to trigger API requests (e.g., fetch users, posts, or random data) with a simple button click. The fetched response is processed asynchronously and immediately displayed using dynamic HTML rendering.

---

## ⚙️ Tech Stack

| Technology | Purpose |
| ---------- | ------- |
| HTML5      | Page layout and structure |
| CSS3       | Styling and hover effects |
| JavaScript | Event handlers and Fetch API |

---

## 🚀 Key Features

* ⚡ **Asynchronous Fetching:** Uses the modern JS Fetch API with async/await handles.
* 🖱️ **Event Handling:** Button click event listeners trigger specific endpoints.
* 🖼️ **Dynamic DOM Rendering:** Modifies target elements dynamically on successful responses.
* 🛡️ **Error Handling:** Safe catch-blocks prevent application crashes on network failure.

---

## 🏗️ Project Structure

* `index.html` - Main user interface with action buttons and data container.
* `main.js` - Contains fetch logic, event listeners, and DOM update functions.
